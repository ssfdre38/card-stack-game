┌─────────────────────────────────────────────────────────────────────┐
│ 1. SAME COLOR                                                       │
│    Any card can be played on a card of the same color              │
│    Example: Red 5 on Red 7 ✅                                       │

┌─────────────────────────────────────────────────────────────────────┐
│ 2. SAME NUMBER (Number Cards Only)                                 │
│    Number cards can match across different colors                  │
│    Example: Red 5 on Blue 5 ✅                                      │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐

│    Action cards can match regardless of color:                     │
│    • Skip on Skip (any colors) ✅                                   │
│    • Reverse on Reverse (any colors) ✅                             │

💻 TECHNICAL IMPLEMENTATION:

The canPlayOn() method in Card.java:

public boolean canPlayOn(Card topCard) {
    // 1. Wild cards can always be played
    if (type == Type.WILD || type == Type.WILD_DRAW_FOUR) {
        return true;
    }
    
    // 2. Match against chosen color after wild
    if (topCard.color == Color.WILD) {
        return true;
    }
    
    // 3. Colors match
    if (color == topCard.color) {
        return true;
    }
    
    // 4. Action types match (Skip, Reverse, Draw Two)
    if (type == topCard.type && type != Type.NUMBER) {
        return true;  // ⭐ This is the key fix
    }
    
    // 5. Number cards match by number
    if (type == Type.NUMBER && topCard.type == Type.NUMBER 
        && number == topCard.number) {
        return true;
    }
    
    return false;
}

Key Line:
  if (type == topCard.type && type != Type.NUMBER)

This allows:
  ✅ Skip to match Skip
  ✅ Reverse to match Reverse
  ✅ Draw Two to match Draw Two
  ❌ But NOT number cards to match just by having same type
