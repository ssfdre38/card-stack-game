⚠️ MANUAL SETUP REQUIRED
═══════════════════════════════════════════════════════════════════════════════

1. 🔐 GitHub Secrets (for automated releases)
   • Navigate to: Settings → Secrets and variables → Actions
   • Add secrets:
     - KEYSTORE_BASE64 (base64 encoded keystore file)
     - KEYSTORE_PASSWORD (your keystore password)
     - KEY_ALIAS (e.g., "matchmania")
     - KEY_PASSWORD (your key password)

2. 📖 Wiki Publication
   • Go to repository Wiki tab
   • Create initial page to enable wiki repo
   • Copy content from wiki/*.md files to GitHub Wiki pages
   • Or clone wiki repo and push content

3. 💬 Discussion Categories
   • Go to Discussions → Categories → Edit
   • Create custom categories (Announcements, Ideas, Q&A, etc.)
   • Configure category settings

4. 🏷️ Repository Topics/Tags
   • Edit "About" section (top right on main page)
   • Add topics: android, game, card-game, ai, open-source, java
